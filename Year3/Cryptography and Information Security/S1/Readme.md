# Guião da Semana 1

## Respostas das Questões

Q1: A versão instalada da biblioteca cryptography é 45.0.5.

## Relatório do Guião da Semana 1

Instalei a biblioteca cryptography usando: pip install cryptography

Verifiquei a versão com: python -c "import cryptography; print(cryptography.__version__)"

Desenvolvi o programa wc.py que conta linhas, palavras e caracteres de um ficheiro.

Executei o programa com: python wc.py exemplo.txt
